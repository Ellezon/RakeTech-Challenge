Assumptions: 
- Single user -> No login required 
- A new task can only be added to the todo
- Tasks can have the same name 

|---------------------------------------------------------------------|

Technologies used:
-PHP
-JS 
-SQL
-HTML
-CSS
-MAMP
-PHPMyAdmin

|---------------------------------------------------------------------|

DB structure:
table name: task
columns: task_ID,task_title,task_notes,task_hours, task_status

Status explained: 
0 = To-Do
1 = In Progress
2 = Done
